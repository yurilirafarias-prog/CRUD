# 📋 Sistema CRUD de Cadastro de Alunos

Este projeto é um sistema simples de CRUD (Create, Read, Update, Delete) desenvolvido em PHP para gerenciamento de cadastro de alunos.

---

## 🚀 Funcionalidades

O sistema permite:

* ✅ Cadastrar alunos
* 📄 Listar alunos cadastrados
* ✏️ Atualizar dados de alunos
* ❌ Excluir registros

---

## 🛠️ Tecnologias Utilizadas

* PHP
* HTML
* CSS
* Banco de Dados (MySQL)

---

## 🧠 Descrição dos Arquivos

* `index.php` → Página inicial do sistema
* `cadastrar_aluno.php` → Lógica de cadastro
* `listar_cadastros.php` → Exibição dos alunos cadastrados
* `atualizar_cadastro_aluno.php` → Atualização de dados
* `form_cadastrar_aluno.php` → Formulário de cadastro
* `form_atualiza_cadastro.php` → Formulário de edição
* `database.php` → Conexão com o banco de dados
* `header.php` / `footer.php` → Layout reutilizável
